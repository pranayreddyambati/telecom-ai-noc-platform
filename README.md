# TelecomAI NOC Platform

AI-powered Telecom Network Operations Center using Python, Flask, Groq LLaMA 3.1, SQLite, and Oracle Cloud Infrastructure.

## Features

- Multi-agent AI pipeline
- Telecom ticket triage
- AI root cause analysis
- SQL agent for customer/network lookup
- NOC-style dashboard
- Oracle-themed UI (planned)
- Planned OCI deployment

## Tech Stack

- Python
- Flask
- Groq API
- LLaMA 3.1
- SQLite
- Oracle Cloud Infrastructure (planned)
- Selenium (planned)
- Behave BDD (planned)

## AI Agents

1. Triage Agent
2. SQL Agent
3. Resolution Agent
